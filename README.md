# Reminder Board

A deadline-focused to-do app built with React + Vite.

Current features:
- Create tasks with a name and deadline.
- Add multiple reminders per task.
- Automatic task grouping into Today, This Week, This Month, and Later.
- Mark tasks as done.

## Local development

```bash
npm install
npm run dev
```
